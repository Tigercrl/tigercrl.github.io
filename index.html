启动器的安装

发表于 2022-07-21 更新于 2022-07-22 Waline：

装完了JDK后，就要安装启动器了。我推荐的话是HMCL和PCL，除此之外还有MultiMC和Minecraft官方启动器，本文讲提供以上列出的启动器安装方法。

启动器的安装

HMCL启动器(Hello Minecraft Launcher)

适用于：Windows MacOS Linux

首先，前往HMCL官网，然后点击右上角的Jenkins CI进入Jenkins页面

点击Permalinks中的Last successful build

然后点击Build Artifacts，再点击(all files in zip)

最后，解压该压缩包并找到HMCL-x.x.x.x.jar，拖到合适的位置并双击打开即可

PCL启动器(Plain Craft Launcher)

适用于：Windows

前往PCL最新下载地址，在下方找到稳定正式版最新的下载链接1并进入

点击普通下载，以下载PCL启动器

下载完成后拖到合适的位置并双击打开即可

MMC(MultiMC)

适用于：Windows MacOS Linux

我的世界官方启动器

适用于：Windows MacOS Linux | 仅基岩版：iOS iPadOS Android

完成！

JDK的安装

发表于 2022-07-21 更新于 2022-07-22 Waline：

从Minecraft1.17开始，Minecraft要求电脑需要JDK16以上，而1.18更是要JDK17以上。所以，在安装游戏之前，需要先安装最新版的JDK。

JDK的安装

在我写这篇文章的时候，最新的JDK是18，那么就给大家演示JDK18的安装

下载安装包

在甲骨文Java官网可以下载到最新版的JDK

Windows系统请选择在Windows分栏里的x64 Installer或者x64 MSI Installer下载

Intel芯片的Mac系统请选择x64 DMG Installer

Apple芯片的Mac系统请选择Arm 64 DMG Installer

Linux系统请根据自己的系统来选择下列框选的安装包

选择完毕后请点击右侧链接进行下载

安装JDK

安装完成后，找到安装包文件

  • Windows系统可以直接打开安装包文件
  • Mac系统需打开DMG文件，再点击其中的.app文件
  • Linux系统我也不太清楚，可以自己上网搜一下

打开后按照提示同意许可后即可下载

完成！
