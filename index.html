Minecraft 安装・启动器的配置

发表于 2022-07-27 Waline： 本文字数： 422 阅读时长 ≈ 1 分钟

装完了启动器后，是不是对启动器的功能非常疑惑？今天我来带大家认识一下 HMCL 和 PCL 的全部功能！

Minecraft 安装・启动器的配置

HMCL 启动器

HMCL 启动器主要分为 3 个板块：账户、游戏和设置，让我们依次讲一下吧！

账户

HMCL 支持 4 种账户的登陆：离线账户、外置账户、Mojang账户和微软账户

进入

点击 HMCL 左上角的玩家头像，即可进入账户页面

离线账户

离线账户是一种免费的账户，俗称盗版账户，不能拥有皮肤。而且 ID 可以随便设

所以可以在某些支持离线账户的服务器上盗别人的号或者被盗号，所以一般这种服务器会添加密码插件

离线模式可以基本上玩到 Minecraft 的所有内容，但是不能进入正版服务器和使用 Realms

添加账户

点击左侧的离线模式

输入你想要的 ID 即可

还可以自定义 UUID（不会不要乱动）

外置账户

外置账户和离线模式一样，免费，可随便设 ID，但是能拥有皮肤。

设置皮肤站

外置账户的登陆需要皮肤站设置，我以后会单独出一期设置皮肤站的文章

添加认证服务器

Minecraft 安装・启动器的安装

发表于 2022-07-21 更新于 2022-07-27 分类于 Minecraft Waline： 本文字数： 830 阅读时长 ≈ 1 分钟

装完了 JDK 后，就要安装启动器了。我推荐的话是 HMCL 和 PCL，除此之外还有 MultiMC 和 Minecraft 官方启动器，本文讲提供以上列出的启动器安装方法。

Minecraft 安装・启动器的安装

HMCL 启动器 (Hello Minecraft Launcher)

启动器简介

系统：Windows MacOS Linux
作者：huanghongxun

首先，前往 HMCL 官网，然后点击右上角的 Jenkins CI 进入 Jenkins 页面

点击 Permalinks 中的 Last successful build

然后点击 Build Artifacts，再点击 (all files in zip)

最后，解压该压缩包并找到 HMCL-x.x.x.x.jar，拖到合适的位置并双击打开即可

PCL 启动器 (Plain Craft Launcher)

启动器简介

系统：Windows
作者：龙腾猫跃

前往 PCL 最新下载地址，在下方找到稳定正式版最新的下载链接1 并进入

点击普通下载，以下载 PCL 启动器压缩包

最后，解压该压缩包后可得到 PCL 启动器，拖到合适的位置并双击打开即可

MMC(MultiMC)

启动器简介

系统：Windows MacOS Linux
作者：MultiMC 团队

前往 MMC 官网的下载栏目，根据自己的系统选择安装包

下载完成后

Windows 系统解压 zip 后运行其中的 MultiMC.exe 即可

Mac 系统解压.tar.gz 后将 MultiMC.app 拖到合适的位置并双击打开即可

注：Linux 的 MMC 需要 QT5 来运行，请自行下载

我的世界官方启动器

启动器简介

系统：Windows MacOS Linux
作者：Mojang Microsoft

前往 Minecraft 官网启动器下载页面

点击页面上方的下载适用于<系统>平台的启动器

若系统不对，可在下方找到适用于其他系统的下载链接

下载完成后

Windows 系统下载后双击并进行安装即可

Mac 系统打开 DMG 后将 Minecraft.app 拖入旁边的 Applications 即可

完成！

Minecraft 安装・JDK 的安装

发表于 2022-07-21 更新于 2022-07-27 分类于 Minecraft Waline： 本文字数： 429 阅读时长 ≈ 1 分钟

从 Minecraft1.17 开始，Minecraft 要求电脑需要 JDK16 以上，而 1.18 更是要 JDK17 以上。所以，在安装游戏之前，需要先安装最新版的 JDK。

Minecraft 安装・JDK 的安装

在我写这篇文章的时候，最新的 JDK 是 18，那么就给大家演示 JDK18 的安装

下载安装包

在甲骨文 Java 官网可以下载到最新版的 JDK

Windows 系统请选择在 Windows 分栏里的 x64 Installer 或者 x64 MSI Installer 下载

Intel 芯片的 Mac 系统请选择 x64 DMG Installer

Apple 芯片的 Mac 系统请选择 Arm 64 DMG Installer

Linux 系统请根据自己的系统来选择下列框选的安装包

选择完毕后请点击右侧链接进行下载

安装 JDK

安装完成后，找到安装包文件

  • Windows 系统可以直接打开安装包文件
  • Mac 系统需打开 DMG 文件，再点击其中的.app 文件
  • Linux 系统我也不太清楚，可以自己上网搜一下

打开后按照提示同意许可后即可下载

完成！
