Minecraft安装 · 启动器的安装

发表于 2022-07-21 更新于 2022-07-23 分类于 Minecraft 本文字数： 784 阅读时长 ≈ 1 分钟

装完了JDK后，就要安装启动器了。我推荐的话是HMCL和PCL，除此之外还有MultiMC和Minecraft官方启动器，本文讲提供以上列出的启动器安装方法。

Minecraft安装 · 启动器的安装

HMCL启动器(Hello Minecraft Launcher)

适用于：Windows MacOS Linux

首先，前往HMCL官网，然后点击右上角的Jenkins CI进入Jenkins页面

点击Permalinks中的Last successful build

然后点击Build Artifacts，再点击(all files in zip)

最后，解压该压缩包并找到HMCL-x.x.x.x.jar，拖到合适的位置并双击打开即可

PCL启动器(Plain Craft Launcher)

适用于：Windows

前往PCL最新下载地址，在下方找到稳定正式版最新的下载链接1并进入

点击普通下载，以下载PCL启动器压缩包

最后，解压该压缩包后可得到PCL启动器，拖到合适的位置并双击打开即可

MMC(MultiMC)

适用于：Windows MacOS Linux

前往MMC官网的下载栏目，根据自己的系统选择安装包

下载完成后

Windows系统解压zip后运行其中的MultiMC.exe即可

Mac系统解压.tar.gz后将MultiMC.app拖到合适的位置并双击打开即可

注：Linux的MMC需要QT5来运行，请自行下载

我的世界官方启动器

适用于：Windows MacOS Linux | 仅基岩版：iOS iPadOS Android

前往Minecraft官网启动器下载页面

点击页面上方的下载适用于<系统>平台的启动器

若系统不对，可在下方找到适用于其他系统的下载链接

下载完成后

Windows系统下载后双击并进行安装即可

Mac系统打开DMG后将Minecraft.app拖入旁边的Applications即可

完成！
